To test additional bots place your bots' python files into the bots folder.

To run the program simply double-click on the tank_launcher.py script. (or you may simply double-click the correct verison of .pyc file for your version of python).

Once the program has started:

  "Space Bar" is used to pause and unpause.
  F toggles between full-screen and windowed mode. 
  t prints the tank location ist
  b prints the bullet list
  q toggles the FPS display
  ESCAPE exits the program

After each battle, the results are stored in a file in the results folder.
